# レストラン味彩 - 予約システム付きWebサイト

シンプルな飲食店のホームページと本格的な予約システムです。

## 主な機能

### トップページ
- レスポンシブデザイン
- コンセプト紹介
- おすすめメニュー表示
- 営業時間・アクセス情報

### メニューページ
- ランチメニュー
- ディナーメニュー
- アラカルト
- ドリンクメニュー

### 予約システム
✅ **カレンダーから日付選択**
- 過去の日付・定休日は自動的に選択不可
- 月の切り替え機能
- 視覚的に分かりやすいUI

✅ **時間帯・人数・コース選択**
- ランチ・ディナーの時間帯別表示
- 1〜8名まで選択可能

✅ **4ステップの予約フロー**
1. 日時・人数・コース選択
2. お客様情報入力
3. 予約内容確認
4. 予約完了

✅ **入力バリデーション**
- 必須項目チェック
- メールアドレス・電話番号の形式チェック
- カタカナチェック

✅ **Googleフォーム連携**
- 予約データを自動的にGoogleフォームへ送信
- スプレッドシートで管理
- メール通知機能

## セットアップ方法

### 1. ファイルをWebサーバーに配置

すべてのファイルをWebサーバーまたはローカル環境に配置してください。

### 2. Googleフォーム連携の設定（推奨）

詳しくは [GOOGLE_FORM_SETUP.md](GOOGLE_FORM_SETUP.md) を参照してください。

**簡単な手順：**

1. Googleフォームを作成（9つの質問を追加）
2. フォームIDとentry IDを取得
3. `js/reservation.js` の設定部分を編集

```javascript
const GOOGLE_FORM_CONFIG = {
    formId: 'ここにあなたのフォームIDを貼り付け',
    entries: {
        date: 'entry.123456789',     // ←各entry IDを設定
        time: 'entry.234567890',
        // ... 以下同様
    }
};
```

4. スプレッドシートでメール通知を設定

### 3. カスタマイズ

#### 店舗情報の変更

各HTMLファイルの以下の部分を編集：
- 店舗名
- 住所
- 電話番号
- 営業時間
- 定休日

#### 定休日の変更

`js/reservation.js` の以下の部分を編集：

```javascript
// 予約不可日（定休日）
const closedDays = [3]; // 0=日曜, 1=月曜, ..., 6=土曜
// 例: 水曜・木曜が定休日の場合 → [3, 4]
```

#### 営業時間・予約枠の変更

`js/reservation.js` の以下の部分を編集：

```javascript
const businessHours = {
    lunch: {
        start: '11:30',
        end: '14:30',
        slots: ['11:30', '12:00', '12:30', '13:00', '13:30', '14:00']
    },
    dinner: {
        start: '17:30',
        end: '22:00',
        slots: ['17:30', '18:00', '18:30', '19:00', '19:30', '20:00', '20:30', '21:00']
    }
};
```

#### デザイン・色の変更

`css/style.css` の以下の部分を編集：

```css
:root {
    --primary-color: #d4af37;      /* メインカラー */
    --primary-dark: #b8941f;       /* メインカラー（暗め） */
    --secondary-color: #2c3e50;    /* サブカラー */
    /* ... */
}
```

## 使い方

### ブラウザで開く

1. `index.html` をブラウザで開く
2. 「ご予約はこちら」をクリック
3. カレンダーから日付を選択
4. 時間帯・人数・コースを選択
5. お客様情報を入力
6. 確認画面で内容を確認
7. 予約を確定

### 予約データの確認

#### Googleフォーム連携している場合
1. Googleフォームの「回答」タブを開く
2. または連携したスプレッドシートを開く
3. メール通知が届く

#### ローカル環境の場合
- ブラウザの開発者ツール（F12）を開く
- Console タブで予約データを確認
- または Application > Local Storage で確認

## よくある質問

### Q. Googleフォーム連携は必須ですか？
A. いいえ、必須ではありません。設定しない場合は、予約データがローカルストレージに保存されます。ただし、店舗への通知はされません。

### Q. 予約データを自分のサーバーに保存したい
A. `js/reservation.js` の `submitReservation()` 関数内で、サーバーへのAPI送信処理を追加してください。

### Q. メニューや価格を変更したい
A. `menu.html` を編集してください。

### Q. 予約可能な最大人数を変更したい
A. `reservation.html` の人数選択のselectタグを編集してください。

### Q. カレンダーのデザインを変更したい
A. `css/style.css` の「予約システム」セクションを編集してください。
