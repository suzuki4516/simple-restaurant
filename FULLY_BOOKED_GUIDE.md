# 📅 満席日設定ガイド

予約がいっぱいになった日を予約不可にする方法を説明します。

---

## 🎯 機能概要

- **満席日をカレンダーでグレーアウト表示**
- **「満席」バッジが表示される**
- **クリック不可になる**
- **簡単に満席日を追加・削除できる**

---

## 📝 満席日の設定方法

### ステップ1: reservation.jsを開く

[reservation.js](js/reservation.js) をテキストエディタで開きます。

### ステップ2: 満席日配列を編集

ファイルの上部（39-45行目あたり）に以下のコードがあります：

```javascript
// ===== 満席日設定 =====
// 満席になった日付をここに追加してください
// 形式: 'YYYY-MM-DD' （例: '2025-12-25'）
const fullyBookedDates = [
    // '2025-12-24',  // クリスマスイブ - 満席（例）
    // '2025-12-25',  // クリスマス - 満席（例）
];
```

### ステップ3: 満席日を追加

配列内に満席の日付を追加します。形式は **`'YYYY-MM-DD'`** です。

**例：12月24日と25日を満席にする**

```javascript
const fullyBookedDates = [
    '2025-12-24',  // クリスマスイブ - 満席
    '2025-12-25',  // クリスマス - 満席
];
```

**例：複数の日付を満席にする**

```javascript
const fullyBookedDates = [
    '2025-11-20',  // 11月20日 - 満席
    '2025-11-21',  // 11月21日 - 満席
    '2025-12-24',  // クリスマスイブ - 満席
    '2025-12-25',  // クリスマス - 満席
    '2025-12-31',  // 大晦日 - 満席
];
```

### ステップ4: 保存

ファイルを保存します（Ctrl+S または Cmd+S）。

### ステップ5: 確認

ブラウザで [reservation.html](reservation.html) を開いて、設定した日付が満席表示になっているか確認します。

## 🎨 カレンダー上の表示

### 満席日の見た目

- **背景色**: 薄いピンク（#ffe5e5）
- **文字色**: 赤（#cc0000）
- **バッジ**: 右上に「満席」と表示
- **クリック**: 不可

### 凡例

カレンダー下部に以下の凡例が表示されます：

- ⚪ 予約可能
- ⚪ 予約不可（定休日など）
- 🔴 **満席**
- 🟡 選択中

## 💡 使用例

### 例1: 今週末が満席になった

```javascript
const fullyBookedDates = [
    '2025-11-16',  // 土曜日 - 満席
    '2025-11-17',  // 日曜日 - 満席
];
```

### 例2: 年末年始が満席

```javascript
const fullyBookedDates = [
    '2025-12-29',
    '2025-12-30',
    '2025-12-31',
    '2026-01-01',
    '2026-01-02',
    '2026-01-03',
];
```

### 例3: 特定のイベント日

```javascript
const fullyBookedDates = [
    '2025-12-24',  // クリスマスイブディナー - 満席
    '2025-12-25',  // クリスマスディナー - 満席
    '2026-02-14',  // バレンタインデー - 満席
];
```

## 🔄 満席日の解除方法

予約に空きが出た場合は、その日付を配列から削除するか、コメントアウトします。

### 方法1: 削除

```javascript
const fullyBookedDates = [
    // '2025-12-24',  // ← この日付を削除
    '2025-12-25',
];
```

### 方法2: コメントアウト

```javascript
const fullyBookedDates = [
    // '2025-12-24',  // ← この日付をコメントアウト（//を追加）
    '2025-12-25',
];
```

## ⚠️ 注意事項

### 1. 日付形式を正確に

**正しい形式:**
```javascript
'2025-12-25'  // ✅ 正しい
```

**間違った形式:**
```javascript
'2025/12/25'  // ❌ スラッシュは使えません
'12-25-2025'  // ❌ 順序が違います
'2025-12-5'   // ❌ ゼロ埋めが必要（'2025-12-05'）
```

### 2. カンマを忘れずに

複数の日付を追加する場合、各行の最後にカンマ（,）を付けます（最後の行は不要）：

```javascript
const fullyBookedDates = [
    '2025-12-24',  // ← カンマ
    '2025-12-25',  // ← カンマ
    '2025-12-26'   // ← 最後はカンマ不要（あってもOK）
];
```

### 3. 過去の日付は自動的に予約不可

過去の日付は自動的にグレーアウトされるため、満席日に追加する必要はありません。

### 4. 定休日（水曜日）は自動的に予約不可

定休日は別の設定（`closedDays`）で管理されているため、満席日に追加する必要はありません。

## 🚀 便利な使い方

### 一括設定用のツール

大量の日付を追加する場合は、以下のJavaScriptコードをブラウザのコンソールで実行すると便利です：

```javascript
// 12月1日から12月31日までの日付を生成
const dates = [];
for (let day = 1; day <= 31; day++) {
    const dateStr = `2025-12-${String(day).padStart(2, '0')}`;
    dates.push(`    '${dateStr}',  // 12月${day}日`);
}
console.log(dates.join('\n'));
```

コンソールに表示された結果をコピーして、`fullyBookedDates` 配列に貼り付けます。

## 📊 管理のベストプラクティス

### 1. コメントを活用

日付ごとにメモを残しておくと管理しやすくなります：

```javascript
const fullyBookedDates = [
    '2025-12-24',  // クリスマスイブ - 貸切パーティー
    '2025-12-25',  // クリスマス - 予約殺到のため満席
    '2026-01-01',  // 元日 - 休業
];
```

### 2. 月ごとにグループ化

大量の日付を管理する場合は、月ごとにグループ化すると見やすくなります：

```javascript
const fullyBookedDates = [
    // 11月
    '2025-11-20',
    '2025-11-21',

    // 12月
    '2025-12-24',
    '2025-12-25',
    '2025-12-31',

    // 1月
    '2026-01-01',
    '2026-01-02',
];
```

### 3. 定期的に見直し

過去の日付が残っていると配列が肥大化するため、定期的に削除しましょう。

## 🔍 トラブルシューティング

### Q: 満席日を設定したのに表示されない

**確認事項:**
1. ファイルを保存しましたか？
2. ブラウザをリロード（F5）しましたか？
3. 日付形式は正しいですか？（`'YYYY-MM-DD'`）
4. JavaScriptエラーが出ていませんか？（F12 → Console）

### Q: JavaScriptエラーが出る

**よくあるエラー:**

```
Uncaught SyntaxError: Unexpected token
```
↑ カンマやクォートが正しくない可能性があります。

**修正方法:**
- 各行の最後にカンマがあるか確認
- シングルクォート（`'`）で囲まれているか確認
- 括弧（`[]`）が閉じているか確認

### Q: 満席日がクリックできてしまう

**原因:**
JavaScriptエラーが発生している可能性があります。

**確認方法:**
F12 → Console タブでエラーを確認してください。
